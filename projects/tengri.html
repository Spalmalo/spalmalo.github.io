Веб-разработка и мобильное приложение
Tengri wallet
Задача:

Один из крупнейших банков Казахстана обратился к нам с объемным заказом на пакет услуг: создание сайта, мобильной версии сайта, сервиса мобильных платежей. Нашей задачей было элегантное и удобное решение для всех клиентов банка, создание таких сервисов, которыми удобно было бы пользоваться нам самим.

Процесс:

Первой работой по этому заказу стало создание сайта с новым дизайном. Затем мы реализовали его мобильную версию. Следующим этапом стала разработка сервиса мобильных платежей. Большая команда разработчиков включилась в этот проект, чтобы реализовать эту крупную задачу. Объединение всех возможных платежей в один сервис реализуются ими по сей день.
